## ⚙️ What I Build

I specialize in architecting backend systems that handle real-world traffic and business requirements:

- **Architecture:** RESTful APIs, distributed services, and background jobs.
- **Security:** Complex Authentication, Authorization, and RBAC flows.
- **Data:** Database design, indexing, and query performance tuning.
- **Infra:** Linux server administration, Nginx, PM2, Docker, and CI/CD pipelines.
- **Internal Tools:** Admin dashboards and operational workflows.
